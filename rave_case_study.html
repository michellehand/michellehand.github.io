Problem framing

Most personal health data is collected passively but rarely interpreted in context. This project asks whether event-specific details — genre, venue, crowd density, substance intensity, embodiment, and social connection — can explain patterns in sleep and next-day energy.

Behavioral analytics Personal data Recovery

Data structure

The dataset is built at the event level and joined to wearable-derived recovery metrics by date. Key fields include event type, genre, crowd size, density, visual intensity, substance intensity, embodiment, energy before, steps, bedtime, wake time, sleep quality, and next-day mood.

Dashboard design

The dashboard uses compact comparisons to show how event intensity relates to recovery. The visual style intentionally mirrors the subject matter while still keeping the analytical structure readable.

Interpretation

Early findings should be treated as exploratory, not causal. The goal is not to prove one variable “causes” recovery, but to generate better questions and identify patterns worth tracking as the dataset grows.

Insights

What this project shows.

Insight 1: Recovery is contextual.

Sleep and next-day energy do not exist in isolation. The value of the event, the social experience, movement, and emotional state all matter.

Insight 2: Subjective ratings are useful signals.

Variables like embodiment, outfit expression, and notable connection may seem soft, but they capture dimensions that wearable data misses.

Insight 3: The dataset gets stronger over time.

With more events, this can evolve from storytelling dashboard to exploratory modeling project with stronger pattern detection.

Reflection

Why it matters.

This project is both analytical and personal: a way to show that data storytelling does not have to be sterile. It can be rigorous, weird, human, and useful at the same time. Tiny spreadsheet goblin, big feelings, valid methodology.
